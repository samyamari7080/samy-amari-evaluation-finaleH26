# Bibliothèque numérique

Application web de gestion de bibliothèque développée avec React, Express et MySQL, réalisée dans le cadre de l'évaluation finale du cours 420-WA6-AG (Hiver 2026).

## Fonctionnalités

- Consultation de la liste des livres disponibles
- Consultation des emprunts d'un utilisateur (recherche par email)

## Technologies utilisées

- Frontend : React
- Backend : Node.js, Express
- Base de données : MySQL

## Installation et lancement

### 1. Base de données
Importer le fichier `bibliotheque.sql` dans phpMyAdmin.

### 2. Backend
cd server
npm install
node server.js

### 3. Frontend
cd client
npm install
npm run dev
